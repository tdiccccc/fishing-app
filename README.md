# 魚(war)

# 概要
 魚(war)は釣り場や釣り方、日付け、時間帯など記録しながら共有できるだけでなく、
 ゲーム感覚で釣果を上げて点数を競うアプリです。
 
# 制作背景
 よく釣りに行く友人と釣果を競っており、いっそアプリにしてしまおうと考えました。
 ただの釣果の記録だけでなく、釣り方や場所、道具などを共有することで、
 私と友人だけでなく、様々なアングラーの人たちの役に立つアプリにしようと思います。
